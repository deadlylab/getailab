# Code of Conduct

GetAiLab beta community standard: **be direct, be curious, be respectful.**

## Expected

- Constructive feedback on research quality, UX, and reliability
- Credit others' ideas in reviews and loop discussions
- Patience with pre-release rough edges — file issues instead of venting in threads

## Unacceptable

- Harassment, discrimination, or personal attacks
- Sharing others' beta access credentials or private lab data without consent
- Publishing CryptO'Brien proprietary materials outside agreed beta terms

## Enforcement

Beta coordinators may revoke access for violations. Report concerns via your GetAiLab contact or a private GitHub issue labeled `conduct` (visible to maintainers only if using private repo).
